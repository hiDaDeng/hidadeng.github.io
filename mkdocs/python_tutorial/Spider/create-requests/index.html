二、访问方法

requests两种访问方法 ,两者都返回Response对象：

requests常用函数 参数解读 使用频率
requests.get(url, headers, cookies, params，proxies) 发起get访问，返回Response对象 绝大多数网站
requests.post(url, headers, cookies, data, proxies) 发起post访问，返回Response对象 少数网站

resp = requests.get(url, headers=headers)

三、返回Response

Response后面带有的状态码：

  • 2开头表示访问正常
  • 4开头，比如403表示爬虫被网站封锁
  • 5开头表示服务器出问题

Response常用方法

Response对象的方法 作用 使用频率
Response.json() 获得json网页数据 35%
Response.text 获得html网页数据 60%
Response.content 获取二进制数据(如文件、图片、音视频) 5%
Response.encoding 根据charset设置网页数据编码 很少用到的
