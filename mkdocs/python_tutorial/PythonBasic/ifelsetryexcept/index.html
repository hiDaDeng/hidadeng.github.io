打印九九乘法表格

用到的知识点:

  • for循环
  • 字符串format方法
  • print函数(涉及到end参数)
for row in range(1, 10):
    #print(row)
    for col in range(1, row+1):
        formula = '{col}*{row}={res}'.format(col=col, row=row, res=col*row)
        print(formula, end='\t')
    print('')
