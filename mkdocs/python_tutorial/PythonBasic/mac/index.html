一、Python安装

官网 https://www.python.org/

mac自带python2，为了与python2区别，凡是在命令行中使用pip和python，我们都要加上3。

安装成功的标准是==命令行可以调用python3==

$ python3



命令行打开的方法

==command+空格== 启动 ==聚焦搜索Spotlight==，再输入terminal



二、pip3设置

pip3是python的命令行安装工具，可以帮我们安装第三方库。

2.1 更改pip3镜像

为了保证安装的速度和成功率，命令行执行

pip3 config set global.index-url https://mirrors.aliyun.com/pypi/simple/

2.2 使用方法

pip3 install packagename

2.3 第三方库安装方法

  1. pip安装法

  2. 单个库的安装，命令行执行

    • pip3 install 库的名字

  3. 多个库的安装， 命令行执行

    • pip3 install -r requirements.txt

  4. pypi本地安装

  5. 在https://pypi.org/ 搜库，点进去

  6. 找Download files，下载whl或压缩文件到桌面。例如文件名 xxx.whl

  7. 命令行依次执行

    • cd desktop

    • pip3 install xxx.whl

  8. github本地安装（如github项目中存在setup.py文件，可以安装使用）

  9. 下载github项目至桌面，解压

  10. 命令依次执行
    • cd desktop
    • python3 setup.py install



三、Jupyter notebook

3.1 安装

命令行执行

pip3 install jupyter

3.2 调用

命令行执行

jupyter notebook

3.3 常用快捷键

jupyter内快捷键 功能
ESC+A（ESC+B） 当前单元格上(下)新建一个新的Cell
D+D 删除当前单元格
Shift+Enter 执行单元格内的Python代码
ESC+M 单元格由代码模式转为Markdown标记模式

推荐： Markdown语法特别好用，强烈建议学习，顺便安装一个Typora软件。
