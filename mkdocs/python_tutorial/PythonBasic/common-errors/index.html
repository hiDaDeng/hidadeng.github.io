2. 误用 =

= 是赋值操作，而判断两个值是否相等是 ==

    print('Man')

  File "<ipython-input-3-c3ceea5a9004>", line 3
    if gender = '男':
              ^
SyntaxError: invalid syntax

if gender == '男':
    print('Man')

Man

8. 使用字典中不存在的键

在字典对象中访问 key 可以使用 []，

但是如果该 key 不存在，就会导致：KeyError: 'zebra'

spam = {'cat': 'Zophie', 
        'dog': 'Basil', 
        'mouse': 'Whiskers'}

print(spam['zebra'])

---------------------------------------------------------------------------

KeyError                                  Traceback (most recent call last)

<ipython-input-24-92c9b44ff034> in <module>
      3         'mouse': 'Whiskers'}
      4 
----> 5 print(spam['zebra'])


KeyError: 'zebra'

为了避免这种情况，可以使用 get 方法

spam = {'cat': 'Zophie', 
        'dog': 'Basil', 
        'mouse': 'Whiskers'}

print(spam.get('zebra'))

None

key 不存在时，get 默认返回 None

\n
\t
\d

如果涉及路径字符串split - Mac pathstr.split('/') - Win pathstr.split('\\')
