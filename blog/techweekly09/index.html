dvt#

https://github.com/distant-viewing/dvt

The Distant TV Toolkit 由几个 Python 包组成，旨在促进视觉文化的计算分析。 开始使用该工具包的最简单方法是在 Google 的协作 (Colab) 环境中运行该工具包。 这是一项免费使用的服务，允许您以最少的设置在远程服务器上运行 Python 代码。


OpenPifPaf#

  • OpenPifPaf：用于语义关键点检测和时空关联的复合字段

  • OpenPifPaf: Composite Fields for Semantic Keypoint Detection and Spatio-Temporal Association

许多图像分析任务需要准确了解人在图像框架内的位置。 我们可以通过使用关键点检测（远景工具包中包含的另一种算法）来获取有关帧中人物位置的更多信息。 在这里，我们检测图像中人物的关键点（与人体相关的点）
