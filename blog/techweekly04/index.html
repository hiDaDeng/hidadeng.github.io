freepn#

https://www.freepn.org/ 免费点对点的科学上网 {{ < figure src=“img/freepn.png” width=“800” >}}

Sharkle#

点击圆圈，随机打开一个有趣的网站。

{{ < figure src=“img/Sharkle.png” width=“800” >}}


免费的 Python 书籍#

https://github.com/pamoroso/free-python-books

这个仓库收集网上的 Python 免费书籍（英文）。


Charts.css#

https://chartscss.org/

一个只使用 CSS 的图形库（不需要 JS），可以把<table>标签变成折线图、直方图或其他图形。

{{ < figure src=“img/charts.png” width=“800” >}}


focalboard#

https://www.focalboard.com/

桌面的看板软件，类似 Trello，可以用来管理待办事项。

{{ < figure src=“img/focalboard.gif” width=“800” >}}


buzzing#

https://www.buzzing.cc/

用中文浏览国外社交媒体里的热门讨论。母语快速导读， 感兴趣再进原文深度阅读

数据显示：使用母语导读比直接浏览英文导读更容易快速找到感兴趣的内容，继而有效提升英文阅读量。

{{ < figure src=“img/buzzing.png” width=“800” >}}


ray#

https://ray.so/

可以生成漂亮的代码截图，颜色、阴影、背景都可以调整。

{{ < figure src=“img/raycode.png” width=“800” >}}


Archivy#

https://github.com/archivy/archivy/

开源的知识库软件，可以自己本地架设，管理个人笔记。


multiavatar#

https://multiavatar.com/

{{ < figure src=“img/multiavatar.png” width=“800” >}} 一个在线工具，可以根据输入的文字，生成用户头像图片（ avatar）。


best-resume-ever#

https://github.com/salomonelli/best-resume-ever

基于 Web 的简历模板，可以生成网页简历，然后用浏览器打印成 PDF 文件。

配置好该项目后，可以一键同时生成多种风格的简历 {{ < figure src=“img/best-resume-ever.png” width=“800” >}}


personal-management-system#

https://github.com/Volmarg/personal-management-system

一个 Web 开源应用，可以自己搭建服务，将个人信息（笔记、代码事项、密码、日程安排、联系人等等）放在一个地方管理。

{{ < figure src=“img/personalmanagement.jpeg” width=“800” >}}


LinkAce#

https://github.com/Kovah/LinkAce/ 自托管的网络书签应用。 {{ < figure src=“img/linkace.png” width=“800” >}}


pap.er#

https://paper.meiyuan.in/ 一个 Mac 电脑的桌面应用，每天获取来自全球的新鲜精美壁纸。 {{ < figure src=“img/paper.png” width=“800” >}}


SVG Favicon Maker#

https://formito.com/tools/favicon

在线制作 SVG 格式的 Favicon {{ < figure src=“img/svgmaker.png” width=“800” >}}


codedmails#

https://codedmails.com/

提供邮件html模板下载 {{ < figure src=“img/codedmails.png” width=“800” >}}
