## 留言功能

博客使用[utterances](https://github.com/utterance/utterances)调用github仓库资源，用于存储评论数据。评论系统有点慢，需要先有github账户才可以使用。日常大家如果对Python感兴趣，有什么好的想法、资料，欢迎在博客中留言。
![](blog/blogcomments/img/留言.png)
